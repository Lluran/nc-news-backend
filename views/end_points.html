API

localhost:9090/api

GET

Success Response - Status: 200. Content: html page with list of all API endpoints (this page)

Topics

localhost:9090/api/topics

GET

Success Response - Status: 200. Content: Array containing all topic info Objects

Topics/Articles

localhost:9090/api/topics/:topic_slug/articles

GET

Success Response - Status: 200. Content: Array containing all article objects belonging to a given topic parameter

POST

Success Response - Status: 201. Content: An object containing the properties of the posted article

Articles

localhost:9090/api/articles

GET

Success Response - Status: 200. Content: Array containing all article-info-objects

Articles/ID

localhost:9090/api/articles/:article_id

GET

Success Response - Status: 200. Content: Object containing info on requested article

PATCH (using query vote=up||down)

Success Response - Status: 200. Content: Object containing info on requested article with new vote count

Articles/ID/Comments

localhost:9090/api/articles/:article_id/comments

GET

Success Response - Status: 200. Content: Array containing comment-objects for given article

POST

Success Response - Status: 201. Content: Object containing posted comment info

Comments/ID

localhost:9090/api/comments/:comment_id

PATCH (using query vote=up||down)

Success Response - Status: 200. Content: Object containing info on requested comment with new vote count

DELETE

Success Response - Status: 202. Content: Object containing info on deleted comment

Users/Username

localhost:9090/api/users/:username

GET

Success Response - Status: 200. Content: Object containing info about given user
